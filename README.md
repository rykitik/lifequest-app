# LifeQuest

LifeQuest — это mobile-first персональная операционная система жизни.

Это не обычный список задач. Приложение объединяет:
- спокойное планирование без перегруза;
- главный квест, быструю победу и запасной план на день;
- визуальный прогресс и XP;
- трекинг тела, энергии и базовых привычек;
- спокойный финансовый контур;
- спасательный режим для плохих дней;
- Ядро-компаньон в духе JARVIS;
- Центр промптов для работы с внешним ChatGPT без платного API внутри приложения.

Проект проектируется как solo-first продукт: сначала для одного пользователя, которому нужна ясность, опора и мягкий возврат в систему.

## Главная идея

Пользователь не просто закрывает задачи.  
Пользователь стабилизирует и развивает свою жизненную систему.

Основной цикл:

```text
Состояние -> Лучший следующий шаг -> Микро-действие -> Награда -> Восстановление -> Рост
```

## Принципы продукта

- mobile-first;
- один главный шаг в каждом блоке;
- минимум когнитивной нагрузки;
- никакого стыда, наказаний и агрессивных streak-механик;
- восстановление важнее идеальности;
- на экране «Сегодня» не должно быть длинных списков задач;
- MVP использует Центр промптов вместо внутренней AI-интеграции.

## Центр промптов

В MVP приложение не вызывает OpenAI API.

Вместо этого LifeQuest:
- собирает текущий контекст пользователя;
- формирует структурированный промпт на русском языке;
- копирует его в буфер;
- открывает ChatGPT во внешнем окне.

## Запуск через Docker

Production:

```bash
docker compose up --build
```

Открыть:

```text
http://localhost:3000
```

Dev через Docker:

```bash
docker compose -f docker-compose.dev.yml up --build
```

Открыть:

```text
http://localhost:5173
```

## Обычный запуск через npm

Установить зависимости:

```bash
npm.cmd install
```

Запустить dev-сервер:

```bash
npm.cmd run dev
```

Собрать production-бандл:

```bash
npm.cmd run build
```

Проверить линтер:

```bash
npm.cmd run lint
```
