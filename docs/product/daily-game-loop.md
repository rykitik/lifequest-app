# Daily Game Loop

Daily Game Loop — ежедневный цикл LifeQuest: открыть приложение, увидеть состояние системы, сделать один полезный шаг, получить реакцию Companion Core и увидеть небольшой рост профиля.

Это не todo list. Экран Today не должен превращаться в длинный список задач. Его задача — выбрать один главный сигнал дня и снизить трение до первого действия.

## Цикл

1. Пользователь открывает Today.
2. Companion Core показывает состояние системы.
3. Daily Quest предлагает один главный квест дня.
4. Пользователь выполняет действие.
5. Reward фиксирует рост модуля и XP.
6. Companion Core даёт короткую реакцию.
7. System Profile показывает статус маршрута и безопасную веху.

## Как выбирается главный квест

Daily Quest строится поверх `todayNextStep`.

Приоритеты MVP:

- если нет body-сигнала, предложить короткий чек-ин тела;
- если воды мало, предложить +500 мл;
- если телесная база есть, но денежный контур пустой или требует внимания, предложить открыть деньги;
- если пользователь в low energy, предложить восстановление;
- если есть маршрут дня, предложить один focus step;
- если данных мало, предложить собрать первый сигнал.

Daily Quest использует безопасные формулировки. Например, focus-квест в System Profile показывается как "Закрыть один быстрый шаг", а не как приватный текст пользовательской задачи.

## Reward

Daily Quest reward засчитывается один раз в день.

Механика:

- `useTodayStore.dailyQuestCompletion` хранит выполненный daily quest на текущую дату;
- `sourceId` в `applyLifeQuestReward` остаётся вторым idempotency-слоем;
- новый день сбрасывает completion;
- повторное нажатие не дублирует XP, toast или Companion reaction.

Reward copy короткий и спокойный:

- "Квест выполнен · Тело +5";
- "Квест выполнен · Фокус +8";
- "Квест выполнен · Восстановление +8".

## Companion reaction

Companion Core реагирует как системный модуль, не как питомец и не как оценщик.

Допустимые реакции:

- "Шаг принят.";
- "Маршрут укреплён.";
- "Сигнал дня зафиксирован.";
- "Ядро стало стабильнее.";

Реакция должна объяснять, что система получила сигнал, а не оценивать пользователя.

## Связь с System Profile

System Profile показывает компактный блок "Сегодняшний маршрут":

- главный квест;
- статус: ждёт действия или выполнен;
- reward signal.

После completion в recent milestones появляется безопасная веха "Главный квест дня выполнен". Она не должна показывать суммы, названия операций, медицинские детали или приватные raw-задачи.

## Anti-toxicity rules

Daily Game Loop не использует:

- потерю серии;
- провал дня;
- штрафы;
- сравнение с другими;
- лидерборды;
- casino-анимации;
- давление "надо каждый день".

Если пользователь ничего не сделал, copy остаётся мягким:

- "Система ждёт один простой шаг";
- "Можно начать с малого";
- "Один сигнал уже достаточно важен".

## Future stages

- quest chains без бесконечного фарма;
- APK home experience с быстрым входом в Today.
