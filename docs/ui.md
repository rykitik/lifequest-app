# UI-спецификация LifeQuest

## Направление дизайна

LifeQuest должен выглядеть как premium dark sci-fi мобильная операционная система.

Ключевые слова:
- тёмный;
- спокойный;
- футуристичный;
- стеклянный;
- неоновый;
- минималистичный;
- читаемый;
- с опорой на Ядро-компаньон;
- mobile-first.

## Палитра

```css
--bg: #0B1020;
```

## Типографика

Рекомендация:
- заголовки: Inter Tight или Space Grotesk;
- основной текст: Inter;
- числа и статусы: JetBrains Mono или IBM Plex Mono.

## Стиль компонентов

Карточки:
- `rounded-2xl` или `rounded-3xl`;
- полупрозрачные поверхности;
- тонкая рамка;
- мягкое свечение на активных состояниях.

Кнопки:
- большие touch targets;
- primary neon gradient;
- понятные disabled states;
- не делать маленькие критические кнопки.

Анимация:
- Framer Motion;
- мягкий пульс для Ядра-компаньона;
- shimmer для прогресса;
- короткий XP-toast;
- без чрезмерного количества particles.

## Экраны

### Сегодня

Верх:
- Ядро-компаньон;
- приветствие;
- текущее состояние;
- уровень и XP.

Середина:
- выбор состояния;
- главный квест;
- быстрая победа;
- запасной план.

Низ:
- полоса секторов;
- быстрые действия;
- нижняя навигация.

### План

- выбор состояния;
- маршрут дня;
- входящие задачи;
- быстрый ввод;
- chips классификации;
- предложение маршрута.

### Тело

- карточка веса;
- вода;
- шаги и движение;
- тренировка;
- питание;
- быстрое действие.

### Деньги

- баланс;
- прогресс долга;
- ежедневные денежные действия;
- спокойная заметка;
- не использовать агрессивный красный UI.

### Ядро

- эволюция Ядра-компаньона;
- прогресс секторов;
- XP и уровень;
- достижения;
- визуализация базы.

### Спасательный режим

- выбор проблемы;
- одно рекомендуемое действие;
- предпросмотр награды;
- кнопка старта.

### Центр промптов

- карточки промптов;
- предпросмотр сгенерированного промпта;
- кнопка «Скопировать промпт»;
- кнопка «Открыть ChatGPT».
