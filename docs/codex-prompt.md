# Промпт для Codex

Открой этот репозиторий и сначала прочитай `README.md`, `AGENTS.md` и все файлы в `docs/`.

Важно: весь пользовательский интерфейс должен быть на русском языке.  
Не оставляй английские UI-лейблы в кнопках, навигации, карточках промптов, мок-данных, заголовках и пользовательской документации.

LifeQuest — production-ready mobile-first PWA.

## Ограничения

- Не редизайнить продукт.
- Не строить внутреннюю AI-интеграцию через платный API.
- Использовать Центр промптов вместо API.
- Начать с frontend-only mock data.
- Держать код чистым, модульным, typed и масштабируемым.

## Первый deliverable

1. React + Vite + TypeScript setup.
2. Tailwind CSS setup.
3. Zustand stores.
4. React Router navigation.
5. Mobile app shell.
6. Bottom nav: Сегодня, План, Тело, Деньги, Ядро.
7. Экран «Сегодня».
8. Companion Core widget с Framer Motion animation.
9. Rescue modal / Спасательный режим.
10. Prompt Center / Поговорить с Ядром:
   - карточки промптов;
   - генерация промпта;
   - кнопка «Скопировать промпт»;
   - кнопка «Открыть ChatGPT».

## Соблюдать дизайн

- premium dark sci-fi UI;
- soft glassmorphism;
- neon accents;
- mobile-first;
- низкая когнитивная нагрузка;
- не делать перегруженные dashboard;
- не показывать длинный список задач на экране «Сегодня».
