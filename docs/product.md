# Спецификация продукта LifeQuest

## Тезис продукта

LifeQuest — это персональная операционная система жизни.

Приложение помогает:
- планировать без перегруза;
- начинать с малого;
- возвращаться после тяжёлых дней;
- снижать прокрастинацию;
- отслеживать тело и финансы;
- видеть визуальный рост;
- использовать ChatGPT внешне через готовые промпты.

Продукт должен конкурировать с быстрыми отвлечениями тем, что вход в полезное действие тоже становится лёгким и приятным.

## Контекст пользователя

Продукт solo-first. Он проектируется вокруг пользователя, который:
- перегружается от сложного планирования;
- уходит в видео, игры и другие отвлечения;
- хочет больше контроля над работой, телом, деньгами и привычками;
- любит футуристичные интерфейсы, помощников в духе JARVIS и визуальный прогресс;
- нуждается в поддержке и структуре без стыда и давления.

## Продуктовая метафора

Жизнь = база / система.  
Полезные действия прокачивают сектора базы.

Секторы:
- Фокус
- Тело
- Деньги
- Стабильность
- Энергия

## Основной цикл

```text
Проверить состояние -> выбрать маршрут -> сделать микро-действие -> получить награду -> усилить сектор -> восстановиться при необходимости
```

## Основные домены

### Сегодня

Операционная точка входа.

Показывает:
- Ядро-компаньон;
- текущее состояние;
- главный квест;
- быструю победу;
- запасной план;
- быстрые действия.

### План

Лёгкий интерфейс планирования.

Функции:
- быстрый ввод;
- классификация задачи chip-кнопками;
- предложение маршрута;
- развернуть большую задачу на 3 шага;
- выбрать главный квест, быструю победу и запасной вариант.

### Тело

Спокойный контур здоровья и веса.

Функции:
- лог веса;
- вода;
- шаги и движение;
- тренировка;
- питание;
- поддержка при риске срыва позже.

### Деньги

Контур финансового контроля без перегруза.

Функции:
- баланс;
- долг;
- прогресс по долгу;
- ежедневные денежные действия;
- спокойная сводка.

### Ядро

Экран прогресса и эволюции.

Функции:
- XP;
- уровень;
- прогресс секторов;
- эволюция Ядра-компаньона;
- достижения.

## Ядро-компаньон

Роль:
- эмоциональный якорь;
- индикатор прогресса;
- спокойное присутствие ассистента.

Он не:
- питомец;
- человек;
- романтический партнёр;
- терапевт;
- судья.

Визуально это:
- светящееся ядро;
- спокойный AI-центр;
- живой sci-fi элемент интерфейса.

Состояния:
- спокоен;
- сфокусирован;
- мало энергии;
- перегружен;
- восстанавливается;
- эволюционирует.

## Спасательный режим

Цель:
помочь пользователю вернуться к действию, когда он застрял.

Проблемы:
- не могу начать;
- устал;
- залип;
- тревога;
- хочу отвлечься;
- хочу сорваться.

Результат:
- одно короткое действие;
- Recovery XP;
- влияние на нужный сектор.

## Центр промптов

Так как платный API не используется в MVP, ChatGPT выступает как внешний мозг.

Центр промптов должен:
- предлагать карточки промптов;
- собирать контекст приложения;
- генерировать полный промпт на русском языке;
- копировать его в буфер;
- открывать ChatGPT.

Карточки:
- Помоги распланировать день
- Я залип, верни меня в систему
- Мне тревожно, помоги мыслить ясно
- Деньги / финансы
- Отношения / одиночество
- Разобрать мою жизнь

## Критерии успеха

Приложение успешно, если пользователь:
- быстро открывает его;
- быстро понимает следующий шаг;
- делает маленькие действия;
- возвращается после плохих дней;
- видит визуальный прогресс;
- чувствует поддержку без осуждения.
